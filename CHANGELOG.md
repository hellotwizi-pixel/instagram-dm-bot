## [2.0.0] - 2026-06-26

### Added
- Initial project structure with TypeScript
- Webhook handling for Instagram messages
- Google Sheets integration for keyword management
- Keyword matching with case-insensitive search
- Instagram Graph API integration for sending DMs
- Unit tests with Vitest (14 tests)
- Vercel deployment configuration
- Complete documentation (README, QUICKSTART, ARCHITECTURE)

### Features
- Automatic keyword detection from comments
- DM auto-send with private_replies API
- 5-second keyword cache for performance
- Full TypeScript type safety
- Error handling and validation
