# 인스타그램 자동 DM 봇 사용 매뉴얼

## 이 봇이 하는 일

당신의 인스타그램 게시물 댓글에서 **특정 키워드를 자동으로 감지**하고, 해당 댓글을 쓴 사람에게 **비공개 DM으로 미리 정해진 자료와 링크를 자동 발송**해주는 봇입니다.

예를 들어, 댓글에 "가격"이라는 단어가 있으면 자동으로 가격표 링크를 DM으로 보내주는 식입니다.

---

## 1단계: 구글시트에서 키워드와 자료 설정하기

### 1-1. 구글시트 준비

봇이 인식할 **키워드와 답장 자료**는 구글시트로 관리합니다. 먼저 구글시트를 준비하세요.

1. Google Sheets(구글 시트)에서 새 시트를 만듭니다
2. 다음 **4개 컬럼**을 첫 번째 줄에 입력합니다:
   - `keywords` (키워드)
   - `replyText` (답장 메시지)
   - `url` (보낼 자료 링크)
   - `postId` (게시물 ID, 선택사항)

### 1-2. 컬럼별 작성 방법

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| **keywords** | 감지할 키워드들 (쉼표로 구분) | `가격, 비용, 가격표` |
| **replyText** | 자동 발송할 메시지 | `안녕하세요! 가격 정보를 확인하세요.` |
| **url** | 보낼 링크 (Notion, Google Drive 등) | `https://www.notion.so/pricing` |
| **postId** | (선택) 특정 게시물에만 적용할 ID | `17999456919123456` |

### 1-3. 구글시트 예시

바로 따라 쓸 수 있는 예시 시트를 만들어 두었습니다. 이 시트를 "파일 > 사본 만들기"로 복사해서 그대로 운영용으로 쓰시면 됩니다.

**예시 시트 바로가기**: https://docs.google.com/spreadsheets/d/1MtK4CVMkNuuoDWBYGrSVZeiM5xHiXdTSg3qHUXzi_PU/edit

시트에는 "비고" 칸에 각 줄이 왜 그렇게 설정됐는지 설명도 같이 적어놨습니다. (비고 칸은 사람이 보는 설명용이고, 봇은 읽지 않습니다.)

**주의:**
- 각 행에 최소 3개 칸은 꼭 채워야 합니다 (keywords, replyText, url)
- `postId` 칸은 비워두어도 됩니다 (아래 섹션 참조)
- keywords 칸에 여러 단어를 넣을 때 **쉼표로 구분**하세요 (예: `가격, 비용, 가격표`)

### 1-4. 구글시트 URL 등록

봇이 이 시트를 읽을 수 있게, 시트의 ID를 서버 설정에 등록해야 합니다. 
(이 작업은 봇을 설치하신 관리자분께 요청하세요)

구글시트 URL이 이렇다면:
```
https://docs.google.com/spreadsheets/d/1a2b3c4d5e6f7g8h9i0j/edit
```

**`1a2b3c4d5e6f7g8h9i0j`** 부분이 시트 ID입니다.

## 2단계: 게시물별로 다르게 반응하기

### 상황: 같은 키워드를 다르게 처리하고 싶을 때

예를 들어, 5번 게시물에서는 "가격" 키워드에 → **할인 가격표 링크**를 보내고,  
2번 게시물에서는 "가격" 키워드에 → **정가 안내** 링크를 보내고 싶을 수 있습니다.

이럴 때는 **`postId` 칸을 사용**합니다.

### 2-1. postId가 없으면 (모든 게시물에 공통 적용)

```
keywords,replyText,url,postId
"가격,비용",안녕하세요! 정상가는 10만원입니다.,https://example.com/normal-price,
```

→ 모든 게시물에서 "가격" 댓글이 오면 "정상가 정보"를 보냅니다.

### 2-2. postId를 지정하면 (특정 게시물에만 적용)

```
keywords,replyText,url,postId
"가격,비용",와우! 오늘만 50% 할인입니다.,https://example.com/discount,17999456919123456
"가격,비용",안녕하세요! 정상가는 10만원입니다.,https://example.com/normal-price,
```

→ **17999456919123456 게시물**에서는 "할인 정보"를 보내고,  
→ **다른 게시물**에서는 "정상가 정보"를 보냅니다.

### 2-3. 게시물 ID 확인하는 방법

**Instagram 그래프 도구 사용 (권장):**
- Meta 제공 API 도구에서 게시물을 조회하면 ID를 확인할 수 있습니다

**공유 링크에서 확인:**
- 게시물을 공유하면 URL에 ID가 포함되어 있습니다
- 정확한 확인이 필요하면 기술 담당자에게 문의하세요

## 3단계: 키워드 수정하기

### 언제 수정이 필요한가?
- 새로운 키워드를 추가하고 싶을 때
- 자료 링크를 바꾸고 싶을 때
- 답변 메시지를 수정하고 싶을 때

### 수정 방법
1. 구글시트를 열고 필요한 부분을 수정합니다
2. **저장**을 클릭합니다
3. 약 5초 후부터 봇이 새로운 설정을 읽습니다 (자동)
4. 다시 테스트해보세요

**재배포 필요 없음!** 구글시트만 수정하면 자동으로 반영됩니다.

## 4단계: 중요하게 알아두기

### 같은 사람이 같은 키워드로 여러 번 댓글을 달면?

현재 봇은 **중복 방지 기능이 없습니다.**

예를 들어:
- 사용자 A가 댓글에 "가격"이라고 2번 댓글을 달면
- DM도 2번 발송됩니다

이것이 의도적인 설계입니다. (나중에 필요하면 추가 가능)

### 현재 상태: 테스트 계정만 작동

**중요:** 현재 코드와 서버는 정상 작동하지만, **Meta의 심사(앱 검수)를 아직 통과하지 못했습니다.**

- **테스트 계정** → DM 자동 발송 ✅ (작동함)
- **일반 사용자** → DM 자동 발송 ❌ (Meta 검수 필요)

Meta의 앱 검수가 완료되면 일반 사용자 댓글에도 자동 응답이 시작됩니다.  
검수 진행 상황은 기술 담당자에게 문의하세요.

## 5단계: 문제 발생 시 확인 사항

### 봇이 작동하지 않을 때 점검하세요

#### 1. 키워드가 정말 등록되었나?
- 구글시트를 열어 `keywords` 칸에 키워드가 정확히 입력되어 있는지 확인하세요
- **오타가 없는지** 체크하세요 (띄어쓰기, 문장 부호 등)

#### 2. 구글시트 URL이 올바른가?
- 봇 설정에 등록된 시트 ID가 정확한지 확인하세요
- 시트의 접근 권한이 "누구나 볼 수 있음" 상태인지 확인하세요

#### 3. 댓글에 정말 키워드가 포함되어 있나?
- 봇은 **정확한 부분 일치**를 합니다 (완전히 같은 단어가 포함되어야 함)
- 예: "가격이" "가격은" "가격??"에서도 "가격"을 인식합니다

#### 4. 게시물 ID를 사용하고 있다면?
- `postId` 칸이 정확히 입력되었는지 다시 확인하세요
- 특정 게시물에서만 작동하지 않는다면, 그 게시물의 ID가 맞는지 확인하세요

#### 5. 자료 링크가 유효한가?
- 실제로 URL에 접속해서 링크가 정상 작동하는지 확인하세요
- 오류 페이지가 뜬다면 URL을 수정하세요

### 그래도 작동하지 않으면?
- 기술 담당자에게 다음 정보를 함께 전달하세요:
  - 어떤 키워드로 댓글을 달았는지
  - 어느 게시물에서 테스트했는지
  - 구글시트에 등록된 설정이 맞는지

## 부록: 자주 하는 질문 (FAQ)

**Q. 키워드를 여러 개 등록하면 모두 작동하나요?**  
A. 네, `keywords` 칸에 쉼표로 구분해 여러 개를 입력하면 모두 감지합니다.

**Q. 대문자와 소문자를 구분하나요?**  
A. 아니요. "가격", "가격", "가격" 등 대문자 구분 없이 모두 인식합니다.

**Q. 링크 대신 다른 텍스트만 보낼 수 있나요?**  
A. 가능합니다. `url` 칸에 링크 대신 텍스트를 입력할 수 있습니다 (예: 전화번호, SNS 아이디 등).

**Q. 같은 게시물에 같은 키워드를 여러 번 등록하면?**  
A. 봇은 처음 일치하는 규칙만 적용하므로, 첫 번째 규칙이 사용됩니다.

**Q. 게시물별로 먼저 확인되나요?**  
A. 네, 같은 키워드라도 `postId`가 지정된 규칙을 먼저 확인하고, 없으면 공통 규칙을 적용합니다.
